## Building and Running the code
### Siga los siguientes pasos:

1. Instale las dependencias:
  ```bash
  npm i
  ```

2. Compile el código TypeScript:
  ```bash
  tsc
  ```

3. Ejecute la aplicación:
  ```bash
  npm run start
  ```
